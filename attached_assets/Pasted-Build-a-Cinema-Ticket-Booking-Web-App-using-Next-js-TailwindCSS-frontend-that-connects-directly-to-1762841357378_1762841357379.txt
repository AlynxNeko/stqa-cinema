Create project in Supabase.io

In Replit, create .env file with:

NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key


Run:

npm install @supabase/supabase-js


Create a file lib/supabaseClient.js:

import { createClient } from '@supabase/supabase-js'
const supabaseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL
const supabaseAnonKey = process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY
export const supabase = createClient(supabaseUrl, supabaseAnonKey)


Use supabase.from('table').select(), .insert(), .update() etc. for data operations.

For Auth, use:

const { data, error } = await supabase.auth.signInWithPassword({
  email, password
})


For file uploads (payment proof):

await supabase.storage.from('payment_proofs').upload(`proofs/${user.id}-${Date.now()}.jpg`, file)
